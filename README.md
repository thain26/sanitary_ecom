# Sanitary E-Commerce

This is the Sanitary E-Commerce project.

## ⚠️ Security Warning

**DO NOT COMMIT `.env` FILES TO GIT.**
The `.env` files contain sensitive keys like database passwords, email app passwords, and Gemini API keys. Please use `.env.example` to track environment variables template.
